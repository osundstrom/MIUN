Arbetsprocessen

Det första jag gjorde var att skapa en map på min hårddisk, inne i mappen på terminalen körde jag npm-init viliket startar ett nytt projekt. Efter detta kör jag npm install parcel --save-dev, vilket installerar Parcel och säger att de bara ska användas vid utveckling. I package.json fyller jag i att index.html är min source(källfil). Efter detta skapar jag filen index.html som är min källfil.

Nu öppnar jag Visual studio code och kopplar den med Git för att kunna versionshantera arbetet. Här gjorde jag en miss i början och la commits och laddade upp de varje gång till main brach i git. Detta åtgärdade jag med att skapa en ny branch för dev som jag använder vid arbete.

Jag skapade hemsdian utefter de krav som fanns och valde att publicera den med Netlify. Inne på Netlify så väljer jag vilket github repo som jag ska publicer och då väljer jag mitt arbete med brach main.

De som automatisras är att alla bilder görs om från jpeg/jpg fromat till webp format. Alla HTML,JavaScript samt CSS filer minifieras
